# Luces Errantes — Frontend

Frontend del juego multijugador en tiempo real, construido con **Svelte 5 + Vite** y renderizado en **Canvas 2D**.

## Inicio rápido

```bash
# Despliegue completo con Docker (desde la raíz del proyecto):
./deploy.sh

# Desarrollo local:
npm install && npm run dev
```

Ver [`DOCUMENTATION.md`](./DOCUMENTATION.md) para la documentación completa del frontend.  
Ver [`../zen-garden-server/ARCHITECTURE.md`](../zen-garden-server/ARCHITECTURE.md) para la arquitectura general y el flujo de despliegue.
